Значения колонок в файле CSK-UGVI-raw.csv:

территория = имя территории
год = год
цск.чн.LOCALITY.GENDER = численность населения на начало года по ЦСК
угви.чн.LOCALITY.GENDER = численность населения на начало года по УГВИ
угви.чр.LOCALITY.GENDER = число рождений за год (УГВИ)
угви.чс.LOCALITY.GENDER = число смертей  за год (УГВИ)
мигр = сальдо миграции (положительная величина означает приток, отрицательная отток)
стаб = если 1, то год включён в стабилизированный участок, иначе пустая строка

LOCALITY может иметь значения:
    гор = города
    сел = уезды
    горсел = города и уезды
    
GENDER может иметь значения:
    м = мужчины
    ж = женщины
    мж = оба пола

*****************************

Файл содержит данные ЦСК и УГВИ без коррекции (кроме исправления опечаток выявленных по перекрёстным суммам в издании),
без слияния крупных городов с губерниями, в которых они находились,
и без слияния губерний и областей образованных после 1897 года с губерниями, из которых они были выделены. 

Данные для Выборгской губернии добавлены по финской статистике.
